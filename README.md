# capstone

A project created for the Capstone at PerScholas.
The purpose of this project is an online learning platform where
users can purchase and complete courses.

## Built With

### Front End
- HTML5,
- CSS,
- Javascript,
- Thymeleaf

### Backend
- SpringBoot,
- SpringSecurity,
- Java,
- MariaDB
